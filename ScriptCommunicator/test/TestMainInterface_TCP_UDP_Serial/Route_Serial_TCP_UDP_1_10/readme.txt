
This script routes:
- serial port (RS232, USB to serial)<->TCP (Client/Server)
- serial port (RS232, USB to serial)<->UDP
- TCP (Client/Server)<->UDP

To execute this script:
- download ScriptCommunicator (http://sourceforge.net/projects/scriptcommunicator)
- configure the first interface (ScriptCommunicator settings dialog) and connect
- add this script in the script window
- start this script, configure the second interface and connect
